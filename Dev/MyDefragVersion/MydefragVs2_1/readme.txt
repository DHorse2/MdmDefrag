===================
  MyDefragGUI 2.1
===================


-------------------
What is It?
-------------------
It is a Graphical User Interface (GUI) for the defragmentation and optimisation program MyDefrag.

---------------------------
Minimum System Requirements
---------------------------
- Windows NT 5.0 (Windows 2000)
- Internet Explorer 6.0
- Pentium II processor with 233 MHz
- 128 MB RAM (~112 MyDefrag, ~16 MyDefragGUI)
- 4 MB of free drive space

-------------------
Installation
-------------------
There is no installation necessary. MyDefragGUI is ready to run and you can use it portable.
Unzip the archive and run MyDefragGUI.exe with double click. You must maintain the folder structure.

-------------------
Bugs
-------------------
If you find bugs or alleged bugs let it me know.

-------------------
Translation
-------------------
If you want to have your own language you have to:
- firstly translate all the data in angle brackets "<...>" in the file "_template.lng" which is in the "languages" folder
- secondly rename the file "_template.lng" to your translated language in english e.g. "dutch.lng"
