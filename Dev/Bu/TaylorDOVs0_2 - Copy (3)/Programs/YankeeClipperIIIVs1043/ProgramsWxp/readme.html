That said, on to the description of the program. The Yankee Clipper III (YCIII) is a Windows 9x/NT4.0/ME/2000/2003/XP clipboard memory/stacker that resides in the system tray. It acts as a silent sentry, continually watching the clipboard, waiting for a change in the text contents. If a change is detected, YCIII saves a copy of the clipboard contents in the event you may want to reuse it. Clicking on the little clipboard in the system tray brings the Yankee Clipper to the top or makes it visible if it was hidden. Double-clicking on an item in the list, or selecting with the arrow keys and hitting <Enter> will place that item back in the clipboard.
